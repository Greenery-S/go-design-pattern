# 工厂方法模式

工厂方法模式使用子类的方式延迟生成对象到子类中实现。

将部分属性和方法的设定放到子类中实现，这样可以在不修改父类的情况下，通过增加子类的方式来扩展父类。

`Go` 中不存在继承 所以使用匿名组合来实现. 类似于:

```go
package main

type Son struct {
	Father
}
type Father struct {
	Name string
}
```

## 使用场景

比如,有一种运算父类,可以通过子类来实现加减乘除等运算. 
- 运算父类可以定义一些公用的方法,比如设定参数,获取结果.
- 子类可以实现具体的运算,和自己需要的特殊属性等.
